# Video Files

Place your video file here and name it `live-casino.mp4` (or update the path in `src/components/LiveVideo/LiveVideo.tsx`).

Supported formats: MP4, WebM, OGG

## Quick Test Video Options:

### Option 1: Generate Test Video (Recommended)
1. Open `scripts/create-test-video.html` in your browser
2. Click "Generate Test Video"
3. Download the generated video
4. Save it as `live-casino.mp4` in this folder

### Option 2: Use Your Own Video
Simply place your MP4, WebM, or OGG video file here and name it `live-casino.mp4`.

## Video Specifications (Recommended):
- Format: MP4 (H.264 codec) for best compatibility
- Resolution: 1280x720 or higher
- Frame rate: 30fps
- Duration: Any length (will loop if needed)
